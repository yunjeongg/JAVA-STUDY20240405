# DataType (데이터 타입)
변수를 선언할 때 주어지는 타입에 따라 변수에 저장할 수 있는 값의 종류와 허용범위가 달라진다.
변수를 선언한 후에는 타입을 변경할 수 없기 때문에 변수를 선언할 때 어떤 타입을 사용할 지 고려해야 한다.

## primitive (기본 타입)
Java 가 제공하는 기본 타입은 총 8개가 있다.
```java
// 정수타입
// 1. byte, 메모리사용크기 : 1byte
byte a = 127;

// 2. short, 메모리사용크기 : 2byte
short b = 32767;

// 3. int, 메모리사용크기 : 4byte // 정수리터럴 
int c = 2147483647;

// 4. long, 메모리사용크기 : 8byte
long d = 9223372036854775807L; // 숫자가 int 의 범위보다 크다면 대문자 L을 붙여줘야 한다.
long e = 2147483647; // int 의 범위 내라면 붙이지 않아도 됨
// 5. char, 메모리사용크기 : 2byte
// - 하나의 문자를 작은따옴표(') 로 감싼 것을 문자리터럴이라 한다.
// 문자리터럴은 유니코드로 변환되어 저장된다.
// 유니코드는 세계각국의 문자를 2byte 로 표현할 수 있는 숫자(0~65535)로 매핑한 국제표준규약이다.
// 자바는 이러한 유니코드를 저장할 수 있도록 2byte 크기인 char 타입을 제공한다.
char var1 = 'A'; // A의 유니코드 : 65 
char var2 = '가'; // 가의 유니코드 : 44032

// 유니코드는 정수이므로 char 도 정수타입에 속한다.
// 그래서 10진수, 16진수 형태로 유니코드를 저장할 수도 있다.
char var3 = 66; // 10진수, 유니코드 66은 B
char var4 = 0x0041; // 16진수

// 작은따옴표(') 로 감싼 문자리터럴은 유니코드로 변환되기 때문에 int 타입 변수에도 저장할 수 있다.
// char 타입 변수에 저장하면 문자열로 매핑되어 출력되지만
// int 타입 변수에 저장하면 유니코드 자체가 출력된다.
char var5 = 'A'; // A가 출력된다.
int var6 = 'A'; // 65가 출력된다.

// 5-1. String 타입 (기본타입 아님!!)
// 작은따옴표(') 로 감싼 문자는 char 타입 변수에 저장되어 유니코드로 저장되지만
// 큰 따옴표로(") 로 감싼 문자 또는 여러개의 문자들은 유니코드로 변환되지 않는다.

// Java 에서 큰따옴표(") 로 감싼 문자들을 문자열이라고 한다.
// 문자열을 변수에 저장하고 싶다면 String 타입을 사용해야 한다.
String var7 = "A";
String var8 = "홍길동";

// 실수타입
// 6. float, 메모리사용크기 : 4byte
// 실수리터럴을 float 타입으로 저장하고 싶다면 뒤에 소문자 f, 또는 대문자 F 를 붙여줘야 한다.
float f = 3.14f;
// 7. double, 메모리사용크기 : 8byte
// 기본적으로 메모리리에 여유가 있고 특별한 이유가 없다면 실수리터럴을 저장시에는 double 을 사용한다.
// Java 의 기본적인 실수리터럴 타입이다.
double g = 3.14;

// 논리타입
// 8. boolean, 메모리사용크기 : 1byte
// Java 는 참과 거짓을 의미하는 논리리터럴로 true 와 false 를 사용한다.
// boolean 타입 변수는 주로 두 가지 상태값에 따라 조건문과 제어문의 실행흐름을 변경하는데 사용한다.
boolean stop = true;
boolean state = false;
```
